UPGradeMind is a web-based polyolefin intelligence console developed at Zhejiang University. It supports polymer R&D workflows through grade data management, predictive models, knowledge graph access, and process simulation modules.

**Key modules**

- Grade database and structured material data management
- Predictive modeling workflows for polymer property and process analysis
- Knowledge graph support for connecting material, process, and application knowledge
- Process simulation tools for polymer intelligence workflows

**Contribution**

System architecture, product workflow design, AI/model integration, and web application implementation.
